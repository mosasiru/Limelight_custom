## 特徴

* 以下のようにLimelight(デフォルトテーマ)をカスタマイズ
  * 時間が常に表示されるように
  * Highlightをとにかくでかく
  * 強調表示のスタイルをItalicで握り潰すのではなく、デフォルトに


## HOW TO USE

```sh
$ cd ~/Library/Application\ Support/Limechat/Themes
$ ln -s /path/to/Limelight_custom/Limelight_custom*
```
